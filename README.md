# Personal Portfolio

A clean, responsive portfolio starter for projects, skills, and contact links.

## Customize

- Edit `index.html` to change your name, bio, project descriptions, and links.
- Edit `styles.css` to adjust colors, spacing, and typography.
- Replace `hello@example.com`, GitHub, and LinkedIn URLs with your real links.

## Open Locally

Open `index.html` in a browser, or run a tiny local server:

```bash
cd portfolio
python3 -m http.server 5173
```

Then visit `http://localhost:5173`.

## Publish With GitHub Pages

Best option for a personal portfolio:

1. Create a GitHub repository named `ssilwalcode.github.io`.
2. Put the contents of this `portfolio` folder at the root of that repository.
3. Push to GitHub.
4. Your public link will be `https://ssilwalcode.github.io/`.

Alternative if you publish inside an existing repository:

1. Push this folder to GitHub.
2. In the repository settings, enable Pages from the main branch.
3. The link will usually look like `https://ssilwalcode.github.io/REPOSITORY-NAME/portfolio/`.
